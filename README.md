# HideFace
## Installation
To install and use this program u should make all requirements:
```bash
pyvenv ./venv
. venv/bin/activate
pip install opencv-python pyqt5 
python hideface.py
```

Or if u using Windows:
```bash
python -m venv ./venv
venv\Scripts\activate.bat
pip install opencv-python pyqt5 
python hideface.py
```
